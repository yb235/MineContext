## 📖 About MineContext

MineContext is an open-source, proactive context-aware AI partner dedicated to bringing clarity and efficiency to your work, study, and creation. It utilizes a context-engineering framework to:

- 📥 **Effortlessly collect** massive amounts of context from various sources
- 🚀 **Proactively deliver** key information and insights
- 💡 **Intelligently resurface** relevant context during creation
- 🛡️ **Ensure privacy** with local-first data storage

## 🏗️ Technology Stack

- **Backend**: Python 3.10+, FastAPI
- **Storage**: SQLite, ChromaDB, VikingDB
- **LLM Integration**: OpenAI, Doubao (ByteDance)
- **Web Interface**: HTML/CSS/JavaScript with Bootstrap
- **Context Processing**: Custom pipeline with chunking, entity extraction, and merging
